Partner story

Replace ad inventory with a proof-backed offer loop.

A partner does not need to believe in a new ad format. They need to see a safe loop: a brand publishes an offer, a buyer mandate limits the agent, the agent redeems only when eligible, and a verifier decides whether the brand pays.

Current claim: production-pilot surface. Partner-ready pilot language remains gated by broad Cowork review; full production-ready still has live/provider blockers.
Brand Publish a budget-capped offer.

Brand teams define sponsored terms, caps, and incentive value. Spend is counted only after a verifier accepts a receipt.

Agent platform Monetise intent without hidden ads.

Agents discover structured offers over MCP, but buyer mandates decide what can be considered, reserved, or redeemed.

Buyer and verifier Keep control and proof separate.

The buyer can opt out of sponsored influence. The verifier rejects tampered receipts before value is released.

01 Brand publishes

Offer terms, budget, sponsorship, and incentive value become machine-readable.

02 Mandate gates

The buyer's scope, spend ceiling, merchant policy, and sponsorship preference are checked first.

03 Agent redeems

The agent reserves and redeems only when the offer fits the mandate and proof path.

04 Verifier decides

Receipts settle only when integrity checks pass; tampered receipts are flagged.

05 Partner reviews

Reports show verified outcomes, zero impressions billed, and the exact DUAL/payment boundary.

What is deliberately excluded
  • No live DUAL write or public write path is enabled.
  • No payment capture, wallet movement, payout, or settlement occurs.
  • OIDC, custom domain, and full production-ready evidence remain separate gates.

Reviewer walkthrough (≈90s)

Brand merge: Revolv is the market-facing product; OfferMesh remains the protocol engine for tenant auth, MCP, proof receipts, metering, and DUAL sync posture.
  1. Run the autonomous agent — it scores disclosed offers, picks the best within its mandate, redeems, gets verified. Full decision trace.
  2. Run the unsponsored principal — that principal opted out of paid influence; the agent takes no sponsored action. Disclosure made this enforceable.
  3. Narrow mandate — gate blocks on scope/merchant/spend; nothing moves.
  4. Red-team — tampered receipt flags for review; no value released.
  5. Brand report — spend equals verified outcomes; impressions billed = 0.
